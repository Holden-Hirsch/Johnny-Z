HTML Structure

The template is based on bootstrap library, please read more about bootstrap here: http://getbootstrap.com/getting-started/ To edit the template, a very basic knowledge in dealing with bootstrap is required.

The Html file can be edited in any text editor, best example is Adobe Dreamweaver.

Hostio template structure is based on bootstrap layout, the page is separated in rows, every row created like the following figure:


The template is arranged in sections, each section(div) has an id, Ex: "header", "top-content", "features", ...etc

<div id="top-content" class="container-fluid">
...
<div id="about" class="container-fluid">
...
<div id="features" class="container-fluid">
...
<div id="info" class="container-fluid">
...
<div id="message" class="container-fluid">
...
<div id="contact" class="container-fluid">
...

CSS

There are four CSS files in the template. First is "bootstrap.min.css", used for layout — originally in bootstrap, the other files are "font-awesome.min.css", "slick.css" and "style.css" is used for style customisation, where the texts, colors, backgrounds and font styles can be changed.

We arrange the styles according to the order of the tags in the html. 

/*------------------------------------------------------------------
[Table of contents]

JavaScript

This template imports four Javascript files.

  • "jquery.min.js": jQuery is a Javascript library that greatly reduces the amount of code that you must write.
  • "bootstrap.min.js": Bootstrap is the most popular Javascript framework for developing responsive, mobile first projects on the web.
  • "slick.min.js": slick is a responsive carousel jQuery plugin that supports multiple breakpoints, CSS3 transitions, touch events/swiping & much more!
  • "main.js": Our custom javascript code.

Works Section

To modify works section , simply edit "data.json" file in the data folder.

For every product there are preview, images and details as follow:

{
    "preview": {
        "img": "images/pro1.jpg"
    },
    "images": {
        "img1": "images/pro1/pro1-prev1.jpg",
        "img2": "images/pro1/pro1-prev2.jpg"
    },
    "details": {
        "title": "Project 1",
        "info": "Lorem ipsum dolor sit amet, consectetur adipiscing elit. Morbi ut vehicula ante. Donec turpis est, pulvinar sed ipsum sed, hendrerit blandit mauris. Cras egestas eget augue vel pulvinar.",
        "skills": "Auctor lobortis
Vel scelerisque
posuere imperdiet",
        "datecompleted": "June 01, 2016",
        "link": "http://brandio.io"
    }
},

You can add any number of images by duplicate this line

"img1": "images/pro1/pro1-prev1.jpg",

and be sure the last one must be without comma at end.



Notice that the first image will be used for preview.

For the details be sure that you don't change these words "title", "info", "skills", "datecompleted" and "link".

The same modifications will be for the popup box that showed after clicking the project.

Form Setup

Set your email in the form:

Open "mailer.php" file in any text editor and replace this email "you@yourdomain.com" with your email, it's in the line 20 of the file.

Replacing Images

To replace the images that not showing as link in the HTML, go to "images" folder, look up for the image that you want to replace and replace it with the same name.
